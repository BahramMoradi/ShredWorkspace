Github repository

  • Create a GitHub account at https://github.com
  • Create a repository
  • Go to created repository settings and find GitHub pages
  • for Source section chose master-branch and save it

GIT

  • in your computer create a folder somewhere
  • Go to the github repositry and copy the clone link of your created repository
  • Open cmd (command line prompet) and navigate to the created folder on your computer
  • Write git clone paste_url_to_your_repository and press enter
  • change dirctory (cd) to downloaded repository in yor folder
  • Now write atom . to open atom IDE in the current project(or folder)
  • Right click on the folder and create new file call it index.html
  • In the index.html file write html and the atom ide will suggest a compilation , enter
  • write some text

push to remote repository

  • open cmd and go to the repository folder
  • write git status and enter. you can se the newly created but untracked files in green color
  • write git add . to add all created file to be tracked
  • write git commit -m "message" to commit the file to local repository
  • To commit to github repository you need to run git push : this command ask you for user name and password of your github account , go ahead and give it the required informations
  • Now your github repository is updatted , you can go to your repository settings on git hub and find the url for the project

    To see the outcome of the change we make to the pages dirctly on the browser, the browser-sync is used.

      browser-sync start --server --dirctory --files "**/*"
      The command start browser sync in server mode (serve up all file in particular dirctory) the --dirctory mode( if it cant find a file the server stil shows a poge with dictory structure )
      --files ( which files watch for changes. if any of thoes files changes though it shows the change on the browser ).
