Useful powershell commands

Finding

  • ls -Recurse -ErrorAction SilentlyContinue | Where-Object{$_.Name -like '*.java'}
  • ls -Recurse -ErrorAction SilentlyContinue *.java

Creating, reading and writing files

  • New-Item test.txt
  • Add-Content 'test.txt' 'The Message'
  • Get-Content test.txt
  • Get-ChildItem -recurse | Select-String -pattern "message" | group path | select name
